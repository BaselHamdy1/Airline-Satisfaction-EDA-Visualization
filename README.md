# ✈️ Airline Satisfaction EDA (Matplotlib)

## 📌 Overview
This project performs **Exploratory Data Analysis (EDA)** on an airline passenger satisfaction dataset using **Python, Pandas, and Matplotlib**.

The main focus is on **data visualization** to understand passenger satisfaction and identify key factors affecting the travel experience.

---

## 📊 Dataset
The dataset contains information about airline passengers, including:
- Demographics (Age, Gender)
- Travel details (Class, Flight Distance)
- Service ratings (Seat comfort, Food, WiFi, etc.)
- Overall satisfaction

Each row represents **one passenger**.

---

## 🎯 Objectives
- Explore the distribution of passenger satisfaction
- Compare satisfaction across different passenger groups
- Analyze service ratings and their impact on satisfaction
- Focus on **seat comfort** as a key factor in customer experience

---

## 🧪 Analysis Steps

### 1️⃣ Satisfaction Distribution
- Visualized the count of satisfied vs dissatisfied passengers
- Helps understand overall airline performance and class balance

---

### 2️⃣ Age Distribution
- Analyzed passenger age distribution using a histogram
- Identified the most common age groups

---

### 3️⃣ Satisfaction by Gender
- Compared satisfaction levels between male and female passengers
- Used grouped bar charts for comparison

---

### 4️⃣ Travel Class Analysis
- Visualized passenger distribution by travel class
- Analyzed satisfaction levels across different classes

---

### 5️⃣ Service Ratings Analysis
- Calculated average ratings for different services
- Compared service ratings between satisfied and dissatisfied passengers

---

### 6️⃣ Seat Comfort Analysis
- Analyzed average seat comfort by travel class
- Studied the relationship between seat comfort and flight distance
- Compared seat comfort ratings between satisfied and dissatisfied passengers

---

## 📈 Tools & Libraries
- Python
- Pandas
- Matplotlib

---

## ✅ Key Insights
- Seat comfort is a major driver of passenger satisfaction
- Premium classes show higher seat comfort ratings
- Seat discomfort becomes more noticeable on longer flights
- Service quality strongly differentiates satisfied and dissatisfied passengers
